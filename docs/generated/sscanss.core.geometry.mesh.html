class sscanss.core.geometry.mesh.Mesh(vertices, indices, normals=None, colour=None, clean=False)¶

Bases: object

Creates a Mesh object. Calculates the bounding box of the Mesh and calculates normals
if not provided. Removes unused vertices, degenerate faces and duplicate vertices when clean is True. The vertices are sorted when clean is performed as a consequence of duplicate removal.
Parameters:
  • vertices (numpy.ndarray) – N x 3 array of vertices
  • indices (numpy.ndarray) – N X 1 array of indices
  • normals (Union[numpy.ndarray, None]) – N x 3 array of normals
  • colour (Colour) – render colour of mesh
  • clean (bool) – flag that specifies mesh should be cleaned

sscanss.core.geometry.mesh.compute_face_normals(vertices, remove_degenerate=False)¶

Calculates the face normals by determining the edges of the face and finding the cross product of the edges. The function assumes that every 3 consecutive vertices belong to the same face. The function can also remove degenerate (zero area) faces.

Parameters:
  • vertices (numpy.ndarray) – N x 3 array of vertices
  • remove_degenerate (bool) – flag that specifies degenerate faces be removed
Returns:

M x 3 array of normals.
Return type:

numpy.ndarray
