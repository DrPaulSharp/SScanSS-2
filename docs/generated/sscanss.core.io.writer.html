sscanss.core.io.writer.write_binary_stl(filename, mesh)¶

Writes a 3D mesh to a binary STL file. The binary STL format only supports face normals while the Mesh object stores vertex normals therefore the first vertex normal for each face is written.

Parameters:
  • filename (str) – path of the stl file
  • mesh (Mesh) – The vertices, normals and index array of the mesh

  • data (numpy.recarray) – 3D points and enabled status
