    sscanss.core.io.reader.read_obj(filename)¶

    Reads a 3D triangular mesh from an obj file. The obj format supports several geometric objects but this function reads the face index and vertices only and the vertex normals are computed by the Mesh object.

    Parameters:filename (str) – path of the obj file
    Returns:The vertices, normals and index array of the mesh
